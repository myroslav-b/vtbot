# VTBot (VirusTotal Telegram Bot)

[English version (Англійська версія) 🇬🇧](README.md)

Telegram бот, який перевіряє вхідні файли за допомогою [VirusTotal API](https://www.virustotal.com/). Бот здійснює пошук файлів за їхнім SHA-256 хешем або завантажує їх для нового аналізу, якщо їх немає в базі даних.

## Функції

- **Перевірка хешу файлу (SHA-256):** Миттєво повертає звіт, якщо файл вже є в базі VirusTotal.
- **Завантаження файлів:** Автоматично завантажує невідомі файли для перевірки.
- **Система черги:** Безпечно обробляє численні запити на перевірку файлів одночасно.
- **Обробка обмежень швидкості:** Враховує ліміти безкоштовного рівня VirusTotal API (4 запити на хвилину) за допомогою вбудованих інтервалів.
- **Обмеження за розміром:** Відхиляє файли об'ємом понад 20 МБ відповідно до можливостей API.
- **Підтримка Docker:** Бот готовий до швидкого розгортання за допомогою Docker та Docker Compose.

## Вимоги

Для запуску цього бота вам знадобиться:
- Telegram Bot API Token (отриманий у [BotFather](https://t.me/BotFather)).
- VirusTotal API ключ.
- Docker та Docker Compose (необов'язково, для розгортання в контейнерах).

## Встановлення та запуск

### Використання Docker Compose (Рекомендовано)

1. Склонуйте репозиторій на свій комп'ютер.
2. Переконайтесь, що у вас встановлені Docker та Docker Compose.
3. Налаштуйте ваші API ключі. Ви можете зробити це, встановивши змінні середовища у терміналі або створивши файл `.env` у кореневому каталозі:
   ```env
   TELEGRAM_TOKEN=your_telegram_bot_token
   VT_API_KEY=your_virustotal_api_key
   ```
4. Зберіть та запустіть бота:
   ```bash
   docker-compose up -d --build
   ```

### Ручне збирання

Якщо ви не хочете використовувати Docker, вам потрібно мати встановлений Go (`1.25` або новішої версії):

1. Експортуйте потрібні змінні середовища:
   ```bash
   export TELEGRAM_TOKEN="your_telegram_bot_token"
   export VT_API_KEY="your_virustotal_api_key"
   ```
2. Зберіть проект:
   ```bash
   go build -o bot ./cmd/bot
   ```
3. Запустіть додаток:
   ```bash
   ./bot
   ```

## Використання

Просто почніть чат з ботом і надішліть йому будь-який документ або файл. Бот прийме його, проаналізує та надішле звіт, у якому буде вказано кількість `шкідливих` (malicious), `підозрілих` (suspicious) та `безпечних` (harmless) спрацювань.
